Eksponen, Logaritma & Pertumbuhan

Eksponen dan logaritma sering muncul pada soal penalaran matematika berbentuk penyederhanaan bentuk, perbandingan nilai, pertumbuhan-populasi, bunga majemuk, hingga model peluruhan. Mengacu pola soal 2025 dan prediksi 2026, topik ini tetap penting karena menguji kombinasi konsep, manipulasi aljabar, dan penalaran konteks.

1. Aturan inti eksponen

Eksponen menyatakan perkalian berulang. Beberapa aturan paling penting:

  • a^m - a^n = a^(m+n)
  • a^m / a^n = a^(m-n), dengan a ? 0
  • (a^m)^n = a^(mn)
  • (ab)^n = a^n b^n
  • a^0 = 1 untuk a ? 0
  • a^-n = 1 / a^n
Contoh: 2^3 - 2^5 = 2^8 = 256. Soal SNBT sering menyamarkan pola ini dalam bentuk pecahan atau akar.

2. Pangkat pecahan dan bentuk akar

Pangkat pecahan sebenarnya hanya cara lain menulis akar:

  • a^(1/n) = nva
  • a^(m/n) = nv(a^m)

Hubungan ini penting untuk menyederhanakan bentuk campuran, misalnya:

16^(3/4) = (2^4)^(3/4) = 2^3 = 8
Trik cepat: pecah bilangan menjadi basis prima lebih dulu. Misalnya 81 jadi 3^4, 32 jadi 2^5, 125 jadi 5^3.

3. Konsep logaritma

Logaritma adalah kebalikan dari eksponen. Jika a^x = b, maka log_a b = x.

Sifat dasar yang harus hafal

  • log_a (MN) = log_a M + log_a N
  • log_a (M/N) = log_a M - log_a N
  • log_a (M^k) = k log_a M
  • log_a a = 1 dan log_a 1 = 0
Catatan: syarat logaritma adalah a > 0, a ? 1, dan M > 0. Pada soal pilihan ganda, syarat ini kadang menjadi sumber jebakan.

4. Aplikasi pertumbuhan dan peluruhan

Soal kontekstual sering memakai model eksponen untuk pertumbuhan populasi, investasi, peluruhan zat radioaktif, atau penyusutan kapasitas.

  • Pertumbuhan: N = N0(1+r)^t
  • Peluruhan: N = N0(1-r)^t atau bentuk setengah umur.
  • Bunga majemuk: nilai akhir terus berkembang berdasarkan hasil sebelumnya.
Contoh: uang Rp1.000.000 tumbuh 10% per tahun selama 2 tahun menjadi 1.000.000 - 1,1- = 1.210.000.

Dalam SNBT, konteks ini kadang disatukan dengan grafik, tabel, atau perbandingan antar skenario.

5. Strategi cepat model SNBT 2025/2026

  1. Ubah semua bilangan ke basis yang sama sebelum membandingkan.
  2. Gunakan sifat eksponen dulu, jangan langsung menghitung angka besar.
  3. Untuk logaritma, ubah ke bentuk eksponen bila lebih mudah dipahami.
  4. Kalau soal konteks, identifikasi dulu apakah itu pertumbuhan atau peluruhan.
  5. Periksa syarat domain pada soal logaritma dan akar.
Mindset ujian: SNBT lebih suka menilai cara berpikir efisien daripada perhitungan panjang. Cari pola, sederhanakan basis, lalu simpulkan.

6. Latihan + pembahasan

Soal 1: Sederhanakan (3- - 35) / 34.

Pembahasan: 3^(2+5-4) = 3- = 27.

Soal 2: Jika log2 32 = x, tentukan nilai x.

Pembahasan: karena 2^5 = 32, maka x = 5.

Soal 3: Sebuah populasi bakteri bertambah 20% setiap jam. Jika mula-mula 500, berapa setelah 3 jam?

Pembahasan: 500(1,2)^3 = 500 - 1,728 = 864. Jadi jumlah bakteri menjadi 864.
